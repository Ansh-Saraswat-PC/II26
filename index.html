Zero-Backend Architecture

Entire solution runs client-side. The scikit-learn pollution classifier is deployed in-browser via ONNX.js for zero-latency inference.

Dual Audience Design

Separate interfaces: Admins receive Auto-generated policy recommendations (e.g., "deploy water sprinklers"), while citizens get plain-language health advisories.

Modern Tech Stack

Built with React, Tailwind CSS, Leaflet.js + GeoJSON, OpenAQ API, and Claude API (Sonnet 4) for natural language generation.
