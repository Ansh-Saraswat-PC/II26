The Crisis

Citywide averages mask deadly micro-zones.

India's cities report a single citywide AQI average. This means residents in high-risk wards—near construction zones, landfills, or biomass-burning areas—receive no targeted warnings.

  • Administrators lack real-time data to deploy corrective action.
  • Current solutions cannot identify the specific source of pollution.
  • Delhi alone sees 54,000+ premature deaths annually linked to air pollution.

The Solution

AirWatch Ward

A real-time, zero-backend platform that analyzes spike patterns and ward metadata to classify pollution sources dynamically.

  • AI Source Detection: Automatically classifies dust, biomass burning, or vehicular emissions.
  • Admin Policies: Auto-generated, ward-specific recommendations.
  • Citizen Advisories: Plain-language health alerts based on localized thresholds.

Key Features

Built for maximum impact and immediate deployment.

Ward-Level Granularity

Goes beyond city averages to pinpoint dangerous micro-zones invisible in state dashboards via interactive Leaflet.js maps.

Claude-Powered Advisories

Natural-language, context-aware policy and health advisories generated on demand—not pre-canned templates.

Policy Impact Preview

Live simulation mode projects AQI improvement if administrator recommendations are followed, turning data into decision confidence.

Zero-Backend Architecture

The entire solution runs client-side. No server infrastructure required for deployment.

React (SPA) Tailwind CSS Leaflet.js + GeoJSON scikit-learn / ONNX.js Claude API (Sonnet 4) OpenAQ API
